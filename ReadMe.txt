----------------------------------------------------------------
SERVEUR VOD - BINOME B3208 - BERTRON AURELIEN / PARISSE ALICIA
----------------------------------------------------------------

		- Description du projet : 
					Création d'un serveur de Vidéo à la Demande gérant plusieurs protocoles
		
		- Protocoles à gérer : 
					-> TCP PUSH
					-> TCP PULL
					-> UDP PUSH
					-> UDP PULL
					-> IP MCAST
	
		- Format d'image supportés : 
					-> BMP
					-> JPEG
